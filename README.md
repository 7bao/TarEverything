# 不用记tar命令参数

## 编译条件

> 1.c++ boost库，Ubuntu下 使用sudo apt install liboost-all-dev命令安装；
>
> 2.make && sudo make install

## 支持解压缩类型

> tar –xvf file.tar
>
> tar -xzvf file.tar.gz
>
> gunzip FileName.gz
>
> tar -xjvf file.tar.bz2 
>
> tar –xZvf file.tar.Z
>
> unrar e file.rar
>
> unzip file.zip 
>
> tar -xvf file.tar.xz
>

## 使用方法

> tareverything file
